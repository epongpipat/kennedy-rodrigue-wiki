# Map Network Directory

To map the `kenrod/` drive (formerly, the `shared/` drive) to your local computer:

::::{tab-set}

:::{tab-item} macOS
Go to `Finder` > `Go` > `Connect to Server`, enter the following and press `Connect`:

```
smb://cvl-share.cvl.utdallas.edu/kenrod
```
:::

:::{tab-item} Windows
Follow the instructions [here](https://atlas.utdallas.edu/TDClient/30/Portal/KB/ArticleDet?ID=51) and enter the following for the `Folder` field:

```
\\cvl-share.cvl.utdallas.edu\kenrod
```
:::

::::
