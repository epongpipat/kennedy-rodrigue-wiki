# Open OnDemand ({{OOD}})

{{OOD}} allows users to access the terminal via a web browser or use programs that require {{GUI}} interfaces (e.g., `SPM`, `freeview`, or `fslview`).

Login using the link below using your {{UTD}} credentials:

<https://ood.cvl.utdallas.edu/>

To use the terminal:

`Clusters` > `>_cortex Shell Access`

To use programs that require a {{GUI}} interface:

`My Interactive Sessions` > `cortex Desktop` > adjust settings as needed and click `Launch`
