# Server

:::{note}
To access our server, make sure that you are connected to the {{UTD}} network:
- If you're on-campus, make sure that you are connected to the CometNet internet
- If you're working off-campus, make sure that you are connected to the {{UTD}} {{VPN}} by following the instructions [here](https://atlas.utdallas.edu/TDClient/30/Portal/Requests/ServiceDet?ID=167)
:::
