# REDCap

REDCap (Research Electronic Data Capture) is a secure web application for building and managing online databases and surveys. We use UTD's REDCap instance to store and manage study-related participant data and information.

## Accessing REDCap

1. Navigate to the official UTD REDCap portal: <https://redcap.utdallas.edu/>
2. Log in using your {{UTD}} NetID and password.
3. Access the study-specific databases and participant logs assigned to your user account.

:::{note}
If you do not have access to the specific study project or need user permissions updated, contact the Study Coordinator or project administrator.
:::
